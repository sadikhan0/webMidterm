# Star Wars Wiki

A React + Tailwind CSS wiki app built with Vite, consuming the [SWAPI](https://swapi.dev) public API.

## Folder Structure

```
src/
├── components/   # Reusable UI components (e.g. Navbar, FilmCard, SearchBar)
├── pages/        # Page-level components (e.g. HomePage, FilmsPage, CharactersPage)
└── services/     # API call functions (e.g. api.js)
```

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS v3
- React Router DOM
- SWAPI (swapi.dev)

## Getting Started

```bash
npm install
npm run dev
```
